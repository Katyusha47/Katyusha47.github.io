Perkenalkan nama saya Muhammad Iqbal Januari. Saya berumur 18 tahun, hobi saya bermain game. Game yang biasa saya mainkan yaitu, CoDM, Battlefield V dll. Biasanya saya bermain game saat hari libur dan jam kosong.

Saya berkuliah di Institut Teknologi Telkom Jakarta. Saya mengambil prodi S1 Teknologi Informasi, baru berjalan satu semester. Saya masuk ke kelas 001 yang berjumlah 27 orang termasuk saya. Selama kuliah saya biasanya mengerjakan beberapa tugas di perpustakaan, bermain game saat jam istirahat dan membuat beberapa proyek. Jarak dari rumah ke kampus sejauh 18KM, jika melewati jalan pintas jaraknya akan lebih pendek yaitu 8KM. Waktu yang ditempuh dari rumah sampai ke kampus adalah 30 menit jika naik motor dan 1 jam jika naik Transjakarta.

Saya memiliki beberapa keahlian, diantaranya menjadi seorang Python Developer, Front-End Web dll. Saya belajar menjadi seorang Front-End web dimulai sejak saya masih bersekolah di SMK. Saat itu semua murid jurusan RPL (Rekayasa Perangkat Lunak) sudah harus bisa menguasai beberapa bahasa pemrograman untuk web seperti PHP, Java dan C#. Setelah memasuki dunia perkuliahan, saya mulai mempelajari bahasa pemrograman Python. Karena dari bahasa pemrograman ini saya bisa mengambil tugas akhir Cyber Security.
